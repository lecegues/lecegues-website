## Usage

### Prerequisites

Make sure you have the following installed:
- **Node.js** (Latest LTS recommended)
- **npm**

### Starting the Server
1. Navigate to project directory
```
cd client
```

2. Start the server
```
npm run start-client
```

3. Open browser and navigate to:
```
http://localhost:3000
```
